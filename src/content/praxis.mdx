## Praxisbeispiel 1

Sie sind der Eigentümer eines Start-Ups und vertreiben Software über einen Webshop im Internet.

Im Rahmen dieser Übung, entwickeln Sie einen Webshop für Ihr Start-Up.

### Aufgabe 0 - Vorbereitung

- Definieren Sie das Softwareprodukt (Businessanwendung, Videospiel, …) welches Sie im Internet vertreiben.
  - Welche Funktionen hat Ihre Anwendung?
    - Upload von mp3- und pdf-Dateien
    - Organisieren der Dateien (z.B. Ordner pro Vorlesungstermin)
    - Zusatzwissen Upload (für große Skripte, für bessere Lernmaterialien, Nutzung von RAG Technologie)
    - Verarbeitung der Daten
      - Transkription der mp3-Dateien mit Whisper-1 auf OpenAI Server
      - Upload des gesamten Kontext (mp3-Transkript, Pdf-Inhalt, Zusatzwissen mit Hilfe von Keyword-Abfrage in der RAG Datenbank)
      - Generierung von Lernmaterialien
        - Zusammenfassung (Kurzfassung der Vorlesung)
        - Veredeltes Skript (Neuformulierung des Transkripts als Fließtext, Entfernung von Füllwörtern, Rechtschreibkorrektur, Strukturierung)
        - Keywords (wichtigste Begriffe der Vorlesung)
        - TLDR (One Pager, 300-500 Wörter)
        - Beispiele (behandelte Beispiele aus der Vorlesung)
        - Begriffe (neue Begriffe mit Definitionen)
        - Anki-Karteikarten (Frage-Antwort Paare für Anki)
    - Lernmaterialien herunterladen (Markdown / PDF, Anki-CSV bereit für Import)
  - Wer ist die Zielgruppe?
    - Studenten
    - Lebenslange Lernende
- Definieren Sie Ihr Unternehmen:
  - Unternehmensname
    - LearnMate
  - Standort
    - Wien, Österreich
  - Anzahl der Mitarbeiter
    - 1

Definieren Sie das Geschäftsmodell (z. B. Abo, Einmalverkauf, Freemium)“

Das Geschäftsmodell ist ein Freemium-Modell mit folgenden Merkmalen:
  - Kostenloses Probekontingent von 100 Tokens.
  - Monatliches Abo für eine feste Anzahl von Tokens (z.B. 10 Euro liefern 3000 Tokens pro Monat).
  - Zusätzlicher Token-Kauf möglich (z.B. 5 Euro für 2000 Tokens).
  - Umrechnung von Tokens müsste noch getestet werden, Stand Dezember 2025 https://platform.openai.com/docs/pricing
    - Transkription mit whisper-1: $0.006 / minute
    - Embedding mit text-embedding-3-large: $0.13 / 1M tokens
    - GPT-5: $1.25 / 1M input tokens, $10.00 / 1M output tokens
    - GPT-5-Mini: $0.25 / 1M input tokens, $2.00 / 1M output tokens

Entwickeln Sie als nächstes einen Webshop, der die folgende Aspekte berücksichtigt:

### Aufgabe 1 - Barrierefreiheit

- Die Website ist WCAG 2.2 AA konform.
- Die Website ist visuell ansprechend und konsistent gestaltet (Farbschema, Typografie, Layout).
- Die Website enthält eine Barrierefreiheitserklärung.
- Hilfreiche Tools: WAVE, AXE DevTools, …

### Aufgabe 2 - Urheberrecht

- Integrieren Sie mehrere Medienarten (z. B. Bilder, Grafiken, Videos, Sounds, Schriftarten, Texte).
- Dokumentieren Sie die Urheberrechtshinweise und Lizenzen aller verwendeten Medien.
- Stellen Sie die Verwertungsrechte der Medien sicher, insb. für:
  - Bilder, Grafiken, etc.
  - Videos, Animationen etc.
  - Musik, Soundeffekte, etc.
  - Schriftarten
  - Texte
- Hilfreiche Tools: Wikimedia Commons, …

### Aufgabe 3 - Marke

- Designen Sie drei unterschiedliche Markenformen (Wortmarke + Wortbildmarke gelten als redundant und zählen daher nur als eine Markenform.)
- Erstellen Sie eine „About us“ Seite, die Ihre Marken und deren Bedeutung erklärt.
- Ordnen Sie Ihre Marken der passenden Nizza-Klassen zu.
- Stellen Sie sicher, dass Sie kein Markenrecht verletzen. Achten Sie insbesondere auf die
  - Wortwahl
  - Bildwahl
  - Farbwahl
  - Positionswahl
  - Formwahl
  - etc.
- Hilfreiche Tools: EUIPO eSearch, Madrid Monitor, TMview, seeip…

### Aufgabe 4 - Lizenz

- Erstellen Sie eine „Lizenz“ Seite
- Definieren Sie welche Nutzungsrechte durch den Kauf erworben werden
- Definieren Sie welche Rechte beim Unternehmen verbleiben
- Formulieren Sie Haftungsbeschränkungen, um Ihre eigene Haftung angemessen zu reduzieren.

### Aufgabe 5 - Datenschutz

- Erstellen Sie eine Datenschutzerklärung.
- Berücksichtigen Sie typische Webshop-Prozesse (z. B. Kontaktformular, Analyse-Tools, Cookies, auch wenn diese auf Ihrer Seite nur gemockt sind).

### Aufgabe 6 - Gewerbe

- Erstellen Sie eine Kontaktseite.
- Erstellen Sie ein Impressum das den Pflichten gemäß § 24 Mediengesetz bzw. den jeweiligen nationalen Vorschriften entspricht.

Die Umsetzung kann in einer Programmier-/Skript-/Markup-Sprache Ihrer Wahl erfolgen.

Ein Backend ist nicht erforderlich. Alle Prozesse dürfen gemockt werden. Eine statische Website ist erlaubt.

Die Abgabe erfolgt über einen Link, wo der Webshop bis zum Ende des Semester aufgerufen werden kann.

Hilfreiche Tools: Github Pages, Cloudflare Pages, …
